# AI Financial News Tracker

Track and analyze financial news with AI.

## Installation

```bash
poetry install
```

## Development

```bash
# Install with dev dependencies
poetry install --with dev

# Run the application
poetry run aifn

# Run tests
poetry run pytest

# Format code
poetry run black .

# Lint code
poetry run ruff check .

# Type check
poetry run mypy aifn
```
